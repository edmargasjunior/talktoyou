Ayuda y Manual de TalkToYou

Este manual explica cómo usar TalkToYou, una aplicación de comunicación alternativa y aumentativa diseñada para ayudar a las personas a expresarse mediante tarjetas con imágenes, textos, sonidos y rutinas.

Objetivo de la aplicación: apoyar la comunicación diaria, permitiendo que la persona toque tarjetas para expresar necesidades, sentimientos, elecciones, molestias, rutinas y situaciones importantes.

1. Pantalla inicial

Al abrir la aplicación, el usuario ve el tablero principal con categorías como:

  • Yo Quiero: necesidades y pedidos inmediatos;
  • Comunicación: respuestas rápidas como sí, no, ayuda y parar;
  • Cómo Estoy: sentimientos, dolor, hambre, sed y malestar;
  • Comer: alimentos sólidos;
  • Beber: líquidos;
  • Rutina: actividades diarias;
  • Sensorial: ruido, luz, molestia y pausa;
  • Emergencia: situaciones importantes o urgentes;
  • Personas: familiares, profesores, terapeutas y amigos;
  • Jugar: juguetes, ocio e intereses.

2. Qué son tarjetas y carpetas

TalkToYou utiliza dos tipos principales de elementos:

Tarjeta

Una tarjeta representa una palabra, acción, sentimiento, objeto o necesidad. Al tocarla, la aplicación habla el texto o reproduce el audio grabado asociado a esa tarjeta.

  • Agua
  • Tengo dolor
  • Sí
  • No quiero
  • Ruido fuerte

Carpeta

Una carpeta es una categoría que contiene otras tarjetas.

  • Yo Quiero
  • Comer
  • Rutina
  • Sensorial

3. Cómo usar el tablero

Paso 1: toque una carpeta principal, por ejemplo Yo Quiero.
Paso 2: toque una tarjeta dentro de ella, por ejemplo Agua.
Paso 3: la aplicación hablará o reproducirá el sonido correspondiente.

Para volver a la pantalla anterior, use la flecha superior de la aplicación o el botón de volver del sistema Android.

4. Cómo funciona la composición de frases

Algunas carpetas pueden combinar su nombre con la tarjeta seleccionada. Esto permite generar frases funcionales simples.

  • Carpeta: Yo Quiero
  • Tarjeta: Agua
  • Frase hablada: Yo quiero agua

Otras carpetas sirven solo para organizar. En estos casos, la aplicación debe hablar solamente la tarjeta seleccionada.

  • Carpeta: Sensorial
  • Tarjeta: Ruido fuerte
  • Frase hablada: Ruido fuerte

Al crear o editar una carpeta, el cuidador puede activar o desactivar la opción: “Combinar el nombre de esta carpeta con la tarjeta seleccionada”.

5. Cómo crear una nueva tarjeta

Paso 1: toque el botón de menú ☰.
Paso 2: elija Agregar Nuevo.
Paso 3: seleccione si el elemento será una Tarjeta o una Carpeta.
Paso 4: elija dónde estará usando el campo Padre.
Paso 5: escriba el nombre de la tarjeta o carpeta.
Paso 6: elija una imagen, grabe audio si desea y toque Guardar en el dispositivo.

6. Cómo editar o eliminar tarjetas

Paso 1: toque el botón de menú ☰.
Paso 2: elija Administrar Ítems.
Paso 3: toque Editar en el elemento deseado.
Paso 4: cambie la información y guarde, o toque Eliminar.
Al eliminar una carpeta, las tarjetas dentro de ella también pueden ser removidas. Use esta opción con cuidado.

7. Cómo agregar una imagen

Al crear o editar una tarjeta, toque Tomar o Elegir Foto.

Según el dispositivo, el usuario puede elegir una imagen de la galería o tomar una nueva foto con la cámara.

Usar imágenes reales del día a día puede mejorar el reconocimiento y el uso funcional.

8. Cómo grabar audio personalizado

TalkToYou permite grabar una voz personalizada para cada tarjeta.

Paso 1: abra el formulario de creación o edición.
Paso 2: toque el botón del micrófono 🎤.
Paso 3: diga el mensaje deseado.
Paso 4: toque nuevamente para detener la grabación.
Paso 5: guarde la tarjeta.

Si una tarjeta tiene audio grabado, la aplicación usará ese audio. Si no, usará la voz sintetizada del dispositivo.

9. Cómo usar la alarma

Cada tarjeta puede tener un horario opcional de alarma.

  • despertar;
  • tomar medicamento;
  • ir a la escuela;
  • hacer tarea;
  • bañarse;
  • dormir.

Cuando llega el horario configurado, la aplicación muestra una pantalla de alarma con la tarjeta correspondiente.

10. Cómo elegir la voz del tablero

En el menú ☰, use la opción Voz del Tablero.

Las voces disponibles dependen del dispositivo, sistema operativo, navegador e idiomas instalados.

11. Cómo cambiar el idioma de la aplicación

TalkToYou puede detectar automáticamente el idioma del celular, pero el cuidador puede elegir manualmente otro idioma en el menú.

Paso 1: abra el menú ☰.
Paso 2: busque Idioma de la Aplicación.
Paso 3: elija portugués, inglés o español.
Importante: las tarjetas personalizadas creadas por el usuario no se traducen automáticamente. Esto protege nombres personales, rutinas, frases terapéuticas y vocabulario específico de la familia. Solo las tarjetas oficiales del sistema pueden ser traducidas.

12. Cómo cambiar el tamaño de las tarjetas

En el menú ☰, use la opción Tamaño de las Tarjetas.

  • Automático;
  • Grande — útil para baja visión o niños pequeños;
  • Mediano;
  • Pequeño;
  • Mini.

13. Copia de seguridad y restauración

TalkToYou permite exportar e importar una copia del tablero.

Exportar copia

Abra el menú ☰ y elija Exportar Copia.

La aplicación generará un archivo JSON con tarjetas, imágenes y audios grabados.

Importar copia

Abra el menú ☰ y elija Importar Copia.

Seleccione un archivo de copia guardado anteriormente.

Al importar una copia, el tablero actual puede ser reemplazado. Exporte una copia antes de importar otro archivo.

14. Borrar datos locales de la aplicación

La aplicación incluye una opción para borrar los datos locales del dispositivo.

  • tarjetas creadas en el dispositivo;
  • imágenes agregadas a las tarjetas;
  • audios grabados;
  • preferencias de voz y diseño;
  • otras configuraciones locales.
Antes de borrar los datos, exporte una copia de seguridad. Esto permite restaurar el tablero posteriormente.

15. Imprimir o generar PDF

En el menú ☰, toque Imprimir Tablero.

Esta función genera una versión en PDF del tablero, útil para imprimir, plastificar o usar como apoyo físico de comunicación.

16. Sugerencia de organización terapéutica

Un tablero útil debe priorizar la comunicación funcional.

  • ¿Qué necesita pedir la persona?
  • ¿Qué necesita rechazar?
  • ¿Cómo puede comunicar dolor o malestar?
  • ¿Cómo puede pedir ayuda?
  • ¿Cómo puede expresar preferencias?
  • ¿Cómo puede participar en la rutina diaria?

Se recomienda personalizar las tarjetas con imágenes reales de la casa, familiares, objetos, alimentos, lugares y actividades que forman parte de la vida de la persona.

TalkToYou puede apoyar diferentes contextos, como autismo, dificultades temporales del habla después de accidentes o enfermedades, y actividades educativas como aprendizaje de vocabulario en otro idioma.

17. Problemas comunes

El audio no se reprodujo

  • Verifique si el volumen del dispositivo está activado.
  • Pruebe otra voz en el menú.
  • Algunos navegadores bloquean el audio hasta el primer toque del usuario.

El micrófono no grabó

  • Verifique si se concedió el permiso del micrófono.
  • Pruebe usar la versión instalada de la aplicación.
  • Algunos navegadores restringen la grabación fuera de contextos seguros.

Mis tarjetas desaparecieron

  • Verifique si los datos de la aplicación o navegador fueron borrados.
  • Restaure una copia de seguridad si está disponible.
  • Evite borrar datos antes de exportar una copia.

La aplicación abrió con una barra del navegador arriba

  • Esto puede ocurrir si se abre como página web y no como app instalada confiable.
  • También puede depender de la configuración PWA/TWA o Android Asset Links.
  • La página HTML no puede ocultar barras de seguridad del navegador.
Consejo: antes de hacer cambios importantes en el tablero, exporte una copia de seguridad.
